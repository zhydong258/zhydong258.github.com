### How do I get set up? ###

* Summary of set up

从GitHub上clone源代码，并初始化和更新submodule。

```
git clone https://github.com/zhydong258/zhydong258.github.com.git blog.github
cd blog.github
git submodule init 
git submodule update
npm install
```
